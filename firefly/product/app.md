# App
*The Firefly mobile application*

The Firefly mobile app is available for iOS and Android, providing a native mobile experience for accessing Firefly's semantic search social media platform.

Currently the app is in early development, with basic connectivity testing and visual branding in place. The app displays the nøøb logo and provides visual feedback about server connection status.

## Platforms

- **iOS**: Native SwiftUI application for iPhone and iPad
- **Android (e/os)**: Native Jetpack Compose application

## Current Features

The app currently implements:
- Server connection monitoring
- Visual connection status (background color)
- Logo display

## Future Development

As Firefly evolves, the app will support:
- User authentication and registration
- Creating and browsing snippets
- Natural language search queries
- Viewing search results with citations
- Offline support and caching
